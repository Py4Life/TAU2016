Homework guidelines

  • The deadline hour is 23:55 on the deadline day.
  • Homework must be submitted via the Moodle system
  • Sumbit an ipynb file, file name: hwX_id.ipynb (replace X with homework number and id with your ID number). For example, the sumbission of homework 5 for a student with ID number 033545885 is hw5_033545885.ipynb
  • Submitted code must run on IPython Notebook with Python 3.5 installed with Anaconda. For instrcutions.
  • Some of the questions are only checked automatically. Hence:
    - Do not change given function names, arguments and types.
    - Make sure that the returned values are exactly as required.
  • Submission will be noted but the code won't always be checked. We will publish a solution after the submission deadline - you must be responsible for verifying your solution.
  • The homeworks will contribute 10% of the final grade, but only if the take-home exam grade is at least 60.
  • Homework is personal and should be solved alone. Students are not allowed to solve homework together, show solutions to other students or read other students' solutions.
  • Homework can be discussed without sharing code. Use the forum in Moodle!
